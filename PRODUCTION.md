## How It Works

The extension monitors your browser activity on supported platforms and displays the Dark Souls "You Died" styled notification whenever configured events occur. Whether it's a task completed, a pull request closed, or any other activity you want to track, experience it in true Souls-like style!

## Browser Support

- Microsoft Edge (143.0+)
- Google Chrome (143.0+)
- Opera Browser (125.0+)

## Privacy

This extension only monitors activity on the websites you visit. It does not collect, store, or transmit any personal data. All processing happens locally in your browser.

# Dark Souls notifications

This project will allow to connect different browser events to a Dark Souls You Died styled message.

## Project Structure

- **Source files**: `src/**/*.ts` (TypeScript)
- **Compiled files**: `src/**/*.js` (generated by TypeScript compiler)
- **Type definitions**: `@types/chrome` for Chrome extension APIs

## Building the Extension

### Install Dependencies

```bash
npm install
```

This will install:
- `typescript` - TypeScript compiler
- `@types/chrome` - Type definitions for Chrome extension APIs
- `@types/node` - Type definitions for Node.js (for build scripts)

### Build TypeScript

```bash
npm run build
```

This compiles all `.ts` files in `src/` to `.js` files in the same location.

### Watch Mode (Development)

```bash
npm run watch
```

This will automatically recompile TypeScript files when they change.

### Clean Build Files

```bash
npm run clean
```

Removes all compiled `.js` files (and `.js.map` source maps).

## Development Workflow

1. Edit TypeScript files in `src/**/*.ts`
2. Run `npm run build` to compile
3. Reload the extension in your browser
4. Test your changes

For faster development, use `npm run watch` in a separate terminal to automatically rebuild on file changes.
